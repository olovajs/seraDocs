🔹 Fragment in SeraJS JSX ​

📦 What is <Fragment>? ​

In SeraJS, <Fragment> is used when you want to return multiple elements without wrapping them in an extra DOM node like a <div>.

✅ Example ​

js
import { h, Fragment } from "serajs";

export default function App() {
  return (
    <Fragment>
      <h1>Hello</h1>
      <p>Welcome to SeraJS</p>
    </Fragment>
  );
}
