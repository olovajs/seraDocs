This guide walks you through how to use setSignal in SeraJS to create reactive state in your UI components.

‚ú® What is setSignal? ‚Äã

setSignal is a function provided by SeraJS that allows you to create reactive state. It returns a getter and a setter ‚Äî when the value changes, the UI updates automatically.

jsx
const [value, setValue] = setSignal(initialValue);
1

value() ‚Äî a function that returns the current value.

setValue(newValue) ‚Äî updates the value and re-renders any part of the UI that depends on it.

üöÄ Example App Here's a basic counter example using setSignal.

jsx
import { h, setSignal, Fragment } from "serajs";

export default function App() {
  const [count, setCount] = setSignal(0);

  return (
    <Fragment>
      <h1>{() => count()}</h1>
      <button onClick={() => setCount(count() + 1)}>Increment Count</button>
      <button onClick={() => setCount(count() - 1)}>Decrement Count</button>
      <button onClick={() => setCount(0)}>Reset Count</button>
    </Fragment>
  );
}
